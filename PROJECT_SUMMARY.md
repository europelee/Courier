## 📊 项目概览

### 项目简介

隧道穿透是一个功能完整的内网穿透隧道管理系统，使用 **Rust + Vue3** 技术栈开发，提供以下核心功能：

- 🌐 **Web 管理界面** - 直观的 Vue3 前端应用
- 🔌 **REST API** - 完整的隧道管理接口
- 🔒 **HTTPS/TLS** - 生产级安全支持
- 📊 **实时监控** - 性能和流量统计
- 🎯 **高效管理** - 搜索、过滤、排序、分页

### 技术栈

| 组件 | 技术 | 版本 |
|------|------|------|
| **后端** | Rust + Axum | 1.94.1 |
| **前端** | Vue3 + TypeScript | 3.x |
| **构建工具** | Vite + Cargo | 4.5.14 + latest |
| **数据库** | SQLite | 3.x |
| **容器化** | Docker Compose | 3.8 |
| **协议** | HTTP/HTTPS/TCP/UDP | RFC 7230+ |
