## 🎨 前端功能详情

### 搜索功能
```vue
<input 
  v-model="searchQuery" 
  type="text" 
  placeholder="搜索隧道（名称、域名、端口）"
  class="search-box"
>
```

**效果：**
- 输入时即时过滤隧道列表
- 支持模糊匹配
- 不区分大小写

### 过滤功能
```vue
<select v-model="filterStatus">
  <option value="">全部状态</option>
  <option value="connected">已连接</option>
  <option value="disconnected">未连接</option>
</select>

<select v-model="filterProtocol">
  <option value="">全部协议</option>
  <option value="http">HTTP</option>
  <option value="https">HTTPS</option>
</select>
```

### 排序功能
```vue
<div class="sort-controls">
  <button @click="sortBy('name')">按名称 {{ sortIcon('name') }}</button>
  <button @click="sortBy('port')">按端口 {{ sortIcon('port') }}</button>
  <button @click="sortBy('created')">按时间 {{ sortIcon('created') }}</button>
  <button @click="sortBy('traffic')">按流量 {{ sortIcon('traffic') }}</button>
</div>
```

### 分页功能
```vue
<div class="pagination">
  <button @click="previousPage" :disabled="currentPage === 1">← 上一页</button>
  <span>第 {{ currentPage }} / {{ totalPages }} 页</span>
  <button @click="nextPage" :disabled="currentPage === totalPages">下一页 →</button>
  <input v-model.number="jumpToPage" type="number" min="1" :max="totalPages">
  <button @click="goToPage">跳转</button>
</div>
```

### 删除确认对话框
```vue
<div v-if="showDeleteConfirm" class="modal-overlay">
  <div class="modal-dialog">
    <h3>确认删除</h3>
    <p>确定要删除隧道 <strong>{{ deleteConfirmTunnel.subdomain }}</strong> 吗？</p>
    <p style="color: #666; font-size: 12px;">此操作不可撤销</p>
    <div class="modal-buttons">
      <button @click="cancelDelete" class="btn btn-secondary">取消</button>
      <button @click="confirmDelete" class="btn btn-danger">确认删除</button>
    </div>
  </div>
</div>
```

### 成功提示
```vue
<div v-if="successMessage" class="success-banner">
  <span>✅ {{ successMessage }}</span>
  <button @click="successMessage = ''" class="close-btn">×</button>
</div>
```

### 加载状态
```vue
<div v-if="isLoading" class="loading">
  <div class="spinner"></div>
  <p>加载中...</p>
</div>
```
